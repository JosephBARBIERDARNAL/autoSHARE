autoSHARE

autoSHARE is a web application that allows researchers to easily access and perform data cleaning/analysis on the SHARE data. Learn more

The application is built using Python 3.10.13 and the Streamlit library (full requirements can be found in the root/requirements.txt file).

Entire source code can be found on the GitHub repository.

You can learn how to install the app on the installation guide.

You can learn more about how the app works on the technical documentation.
